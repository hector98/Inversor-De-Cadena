# String Inversor


A program that reads a string and prints it in reverse order.

## Characteristics

- ✅ Validation of the string
- ✅ Print the string in reverse order with spaces
- ✅ Error management
- ✅ Comand Line Interface (CLI)
- ✅ Unit Testing
- ✅ Documentation
- ✅ JSON output

## Installation
```bash
git clone git@github.com:hector98/Inversor-De-Cadena.git
cd Inversor-De-Cadena

# Install dependencies
bundle install

# Run the program
./bin/app_cli
```
